Widget iOS — 3 étapes

① Installe le script
Installe Scriptable (App Store, gratuit), puis copie le script ci-dessous dans un nouveau script nommé Tâches.


② Exporte tes tâches
Appuie sur le bouton, puis dans la feuille de partage iOS choisis Enregistrer dans Fichiers → iCloud Drive / Scriptable et sauvegarde sous le nom taches.json.


③ Ajoute le widget
Sur l'écran d'accueil, maintiens appuyé → + → cherche Scriptable → choisis la taille → sélectionne le script Tâches. Répète l'étape ② à chaque fois que tu veux rafraîchir le widget.
